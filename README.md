# SimpleNote

SimpleNote は、  
日常のメモを登録し、  
タグで整理できるシンプルなメモアプリです。

Laravel 学習の一環として、  
**Laravel 6 前提の講座を参考にしながら実装し、  
Laravel 12 環境向けに構成やコードを見直して作り直しました。**

メモアプリという題材を通して、  
Laravel における基本的な設計や処理の流れを確認することを目的としています。

---

## 🎯 目的

- Laravel を用いた Web アプリケーションの基本構成を理解する
- CRUD 処理の一連の流れを実装を通して確認する
- 講座の内容をそのまま写すのではなく、自分で構成を整理し直す

---

## 🛠 使用技術

- PHP 8.5
- Laravel 12
- MySQL
- Bootstrap
- Docker（Laravel Sail）

---

## ✨ 主な機能

- ユーザー登録 / ログイン / ログアウト
- メモの作成・編集・削除
- メモ一覧・詳細表示
- タグによるメモ分類

---

## 🧩 実装について

- メモに関する CRUD 処理は **MemoController に集約**しています
- 一覧表示などの画面表示は HomeController に切り分け、  
  Controller ごとの責務が分かりやすくなるよう意識しました

※ 講座を参考にしつつ、  
※ Laravel 12 環境向けに Controller 構成を整理し直しています

---
